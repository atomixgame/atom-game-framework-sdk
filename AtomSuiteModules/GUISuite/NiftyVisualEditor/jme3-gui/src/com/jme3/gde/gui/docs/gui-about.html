About Visual GUI Support

Visual GUI Editor was created with the intent to simplify the creation of your Nifty, ToneGod, Lemur gui. Loosely inspired a previous Nifty-Editor software brings many new features both in code and functionality. First of all is the ability to save and load your gui in xml files which then can be read by your application using Nifty-Gui library. Moreover, the ease of use will save a lot of time not only in the construction of the gui but also in learning xml syntax used by Nifty-Gui. Indeed the software will translate what you see in your MAINVIEW to a xml file without you know nothing of how Nifty-Gui read the file. For now the software it is only in alpha stage but the main functions are already operating without many bugs.
